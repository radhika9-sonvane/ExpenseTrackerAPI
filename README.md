# ExpenseTracker
It is a Web Application bulit in ASP.Net MVC and API with Entity Framework.

## Prerequisites
- Visual Studio 2022
- SQL Server 2018

## NuGet packages
- Entity Framework
- Microsoft.AspNet.WebApi.Client
- AlertifyJs
- jquery

## Features
- Add, edit, and delete expense Details
- Add, edit, and delete category
- Add, edit TotalLimit
